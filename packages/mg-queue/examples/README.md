# Examples

Run any example from the `packages/mg-queue/` directory:

```bash
npx tsx examples/<filename>.ts
```

## verbose-renderer.ts

Runs concurrent tasks with the verbose renderer, which logs every start, complete, and error event.

```bash
npx tsx examples/verbose-renderer.ts
```

## dynamic-renderer.ts

Runs 20,000 tasks through the dynamic renderer with `keepOnScreen`, showing a live-updating spinner with subtask support.

```bash
npx tsx examples/dynamic-renderer.ts
```

## silent-mode.ts

Runs tasks with no renderer. Demonstrates how to inspect `QueueStats` for completed, skipped, and errored tasks.

```bash
npx tsx examples/silent-mode.ts
```

## skip.ts

Demonstrates all skip patterns: boolean skip, sync/async skip functions, and in-execution `ctx.skip()`.

```bash
npx tsx examples/skip.ts
```

## timeout.ts

Configures a queue-wide timeout. Shows how slow tasks fail with `TimeoutError`.

```bash
npx tsx examples/timeout.ts
```

## subtasks.ts

Returns `Subtasks` from parent tasks to run child tasks in parallel or sequentially.

```bash
npx tsx examples/subtasks.ts
```

## abort-signal.ts

Uses `ctx.signal` for cooperative cancellation — aborting fetch-like operations and polling loops when tasks time out.

```bash
npx tsx examples/abort-signal.ts
```

## sqlite-reader.ts

Streams rows from a SQLite database as queue tasks using `SqliteReader`. Requires a database generated by `scripts/generate-sqlite-posts.sh` first.

```bash
npx tsx examples/sqlite-reader.ts
```

## other.ts

A minimal end-to-end example using the dynamic renderer with `keepOnScreen`. Sequential root tasks with parallel subtasks showing task hierarchy.

```bash
npx tsx examples/other.ts
```
